# Product Roadmap

## Phase 1: MVP (Current Focus)
**Goal**: A lightweight, no-auth tool for instant LinkedIn analytics visualization.
- [ ] **Excel Upload**: Browser-based parsing of LinkedIn `.xlsx` export.
- [ ] **Validation**: Strict client-side schema validation.
- [ ] **Dashboard**:
    - Impressions & Engagement over time.
    - Follower growth (if data available).
    - Top Posts table.
- [ ] **Auto-Insights**: Rule-based highlights (e.g., "Best time to post").
- [ ] **No Auth**: Session-based usage only.

## Phase 2: Polish & Export
**Goal**: Enhance utility for power users.
- [ ] **PDF/Image Export**: Download charts for reports.
- [ ] **Enhanced Charts**: Compare metrics (e.g., Impressions vs. Clicks).
- [ ] **Dark Mode**: UI theme toggle.

## Phase 3: Accounts & History (Future)
**Goal**: Long-term tracking.
- [ ] **Authentication**: User accounts to save history.
- [ ] **Multi-upload**: Track growth across multiple export files.
- [ ] **Competitor Benchmarking**: Manual entry comparison.
